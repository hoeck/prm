\section{Allgemeine Projektbeschreibung}

\subsection{Projektthema}

\paragraph*{}
In diesem Projekt soll der personalfreie Zugang und die personalfreie Ausleihe von Medien (Bücher/CDs/DVDs/Zeitungen...) realisiert werden. Vorschläge seitens der Auftraggeber sind die Lösung mittels der RFID-Technologie per Gate oder Terminal. Hierbei soll die ID des Nutzers über eine Karte ermittelt werden. Der Zugang zur Bibliothek soll nur für autorisierte Personen ermöglicht werden. Den Medien sollen ebenfalls per RFID eine ID zugewiesen werden. Wenn die Bücher aus dem Raum gelangen ohne das die ID des Buches mit einer Nutzer ID verknüpft wurde, soll ein Alarm ähnlich wie im Kaufhaus ausgelöst werden. Die ID des Nutzers soll mit der ID des Mediums ans System übertragen werden (Anbindung am Lotus Notes).

Benötigte Rechner oder Server sind bereits vorhanden, eine Internetanbindung ist über WLan gegeben.
